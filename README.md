# Final_project

Установка
---------
```
1. Скачать актуальный webdriver для chrome и задать в переменной среды Path путь к нему
2. Установить python3.11 с официального сайта
3. Клонировать проект из репозитория: git clone https://github.com/RushmanovaOlga/Final_project
4. Перейти в папку с проектом: cd Final_project
5. Создать виртуальное окружение: virtualenv venv
6. Активировать виртуальное окружение: .\venv\Scripts\activate
7. Установить зависимости: pip install -r requirements.txt
```

Запуск
------

```
pytest .\test_rt.py
